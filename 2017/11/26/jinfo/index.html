动态打开jvm的开关

jinfo可以动态的改变jvm的flag， 而不必重启服务器。虽然只对一些特定的flag有效，但是有的时候也很有用。

支持动态开启和关闭的的flag，可以通过下面的命令查看。

java -XX:+PrintFlagsFinal -version | grep managed
